# 本文图片存放说明

将以下图片放入此目录后，文章即可正确显示。

## 图片清单

| 文件名 | 用途 | 建议尺寸 | 建议大小 |
|--------|------|----------|----------|
| `cover.jpg` | 封面图（首页卡片 + 文章顶部） | 1200×630px | < 200KB |
| `collab-tool-1.jpg` | 协作工具指南 图1 | 最大 1200px 宽 | < 150KB |
| `collab-tool-2.jpg` | 协作工具指南 图2 | 最大 1200px 宽 | < 150KB |
| `collab-tool-3.jpg` | 协作工具指南 图3 | 最大 1200px 宽 | < 150KB |
| `collab-tool-4.jpg` | 协作工具指南 图4（如有更多） | 最大 1200px 宽 | < 150KB |

## 图片格式建议

- **格式：** JPEG（照片）或 PNG（截图）或 WebP（体积更小）
- **封面图：** 1200×630px 适配社交分享卡片和首页
- **文内图：** 宽度 ≤ 1200px，单张 < 200KB 为佳

## 优化工具

- [Squoosh](https://squoosh.app/) — 在线压缩
- [ImageOptim](https://imageoptim.com/) — Mac 压缩工具
